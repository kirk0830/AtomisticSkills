# Workflow: Calculate Activation Energy and Diffusion

This workflow describes how to calculate the ionic diffusion coefficient ($D$) at multiple temperatures and derive the activation energy ($E_a$) using Arrhenius and Molecular Dynamics (MD) simulations.

1. **Prepare Structure**:
   - Obtain the initial structure.
   - If the structure has periodic boundary condition, create a supercell large enough (typically > 10 Å in all dimensions) to minimize periodic boundary artifacts and self-interaction. The total number of atoms in the cell should be around few hundred to thousand.
   - Save the structure as `supercell.cif`.

3. **Select and Load MLIP**:
   - Consult `.agent/rules/foundation-potentials.md` to choose the appropriate foundation potential.
   - Use the `load_model` tool to load the selected model.

4. **Run MD Simulations**:
   - Run **NVT** ensembles at multiple temperatures.
   - **Temperatures**: Choose a range below the melting point but high enough to observe diffusion (e.g., 300K, 400K, 500K, 600K, 700K, 800K).
   - **Duration**: sufficient to reach diffusive regime (linear MSD), typically >50 ps (e.g., `steps=50000` with `timestep=2.0` fs gives 100 ps).
   - Use the `run_md` tool for each temperature.
   
   *Example configuration for `run_md`*:
   - `structure_data`: "supercell.cif"
   - `ensemble`: "nvt"
   - `temperature`: [T]
   - `steps`: 50000
   - `timestep`: 2.0
   - `output_dir`: "results/md_[T]K"

5. **Analyze Results**:
   - Use the following Python script (`analyze_diffusion.py`) to calculate MSD, Diffusion Coefficient ($D$), and Activation Energy ($E_a$).
   - This script reads the ASE `.traj` files generated by `run_md`.

   ```python
   import numpy as np
   from ase.io import read
   from scipy.stats import linregress
   import matplotlib.pyplot as plt
   import glob
   import os

   def calculate_msd(images, mobile_indices):
       """
       Calculate Mean Squared Displacement (MSD) for mobile species.
       Handles unwrapping of coordinates.
       """
       positions = np.array([atoms.get_positions() for atoms in images])
       cell = images[0].get_cell()
       
       # Unwrapping coordinates
       # d_pos = pos[t] - pos[t-1]
       # if d_pos > L/2, subtract L; if d_pos < -L/2, add L
       diff = positions[1:] - positions[:-1]
       fractional_diff = np.linalg.solve(cell.T, diff.T).T
       fractional_diff -= np.round(fractional_diff)
       real_diff = np.dot(fractional_diff, cell)
       
       # Integrate to get unwrapped displacements
       unwrapped_positions = np.cumsum(real_diff, axis=0)
       unwrapped_positions = np.vstack([np.zeros_like(positions[0]), unwrapped_positions])
       
       # Calculate MSD for mobile species
       # MSD(t) = <|r(t) - r(0)|^2>
       # simplified: just referencing from t=0
       msd = np.mean(np.sum(unwrapped_positions[:, mobile_indices, :]**2, axis=2), axis=1)
       return msd

   def analyze_diffusion(base_dir, species_element="Li"):
       """
       Analyze diffusion across multiple temperatures.
       """
       traj_files = sorted(glob.glob(f"{base_dir}/*/*.traj"))
       
       temps = []
       diff_coeffs = []
       
       plt.figure(figsize=(10, 6))
       
       for traj_file in traj_files:
           print(f"Processing {traj_file}...")
           try:
               traj = read(traj_file, index=':')
               if len(traj) < 10:
                   print("  Skipping: too few frames")
                   continue
                   
               # Identify mobile species indices
               symbols = np.array(traj[0].get_chemical_symbols())
               mobile_indices = np.where(symbols == species_element)[0]
               if len(mobile_indices) == 0:
                   print(f"  Skipping: species {species_element} not found")
                   continue

               # Extract temperature from path or average kinetic energy
               # Here assuming path format "results/md_800K/..." from workflow
               try:
                   T_str = os.path.dirname(traj_file).split('_')[-1].replace('K', '')
                   T = float(T_str)
               except ValueError:
                   # Fallback: calculate from mean Kinetic Energy if needed
                   T = np.mean([a.get_temperature() for a in traj])
               
               # Calculate MSD
               msd = calculate_msd(traj, mobile_indices)
               
               # Time array
               # Assuming timestep recorded in metadata or approximated?
               # run_md saves logfile. Let's assume standard behavior or read form headers if available.
               # For now, assuming 100fs sampling (run_md default log_interval=100 * timestep=1?)
               # Users should verify `time_step` * `log_interval` from their run_md settings.
               dt_fs = 100.0 # Placeholder, adjust based on run_md settings (e.g. 1fs * 100)
               dt_ps = dt_fs / 1000.0
               time = np.arange(len(msd)) * dt_ps
               
               # Linear fit to get D
               # MSD = 6*D*t + b (3D diffusion)
               # Fit linear part (e.g., last 50%)
               start_idx = int(len(time) * 0.5)
               slope, intercept, r_value, p_value, std_err = linregress(time[start_idx:], msd[start_idx:])
               
               # D in Angstrom^2/ps = 1e-20 m^2 / 1e-12 s = 1e-8 m^2/s
               # D = slope / 6
               D = slope / 6.0 
               # Convert to mS/cm or similar if needed.
               # standard D units: Ang^2/ps
               
               # Convert to cm^2/s
               # 1 Ang^2/ps = (1e-8 cm)^2 / (1e-12 s) = 1e-16 / 1e-12 = 1e-4 cm^2/s
               D_cm2s = D * 1e-4
               
               temps.append(T)
               diff_coeffs.append(D_cm2s)
               
               plt.plot(time, msd, label=f"{T:.0f} K (D={D_cm2s:.2e} cm²/s)")
               
           except Exception as e:
               print(f"Failed to process {traj_file}: {e}")

       plt.xlabel("Time (ps)")
       plt.ylabel("MSD (Å²)")
       plt.legend()
       plt.title(f"MSD for {species_element}")
       plt.savefig("msd_plot.png")
       print("Saved msd_plot.png")
       
       # Arrhenius Plot
       if len(temps) >= 3:
           inv_T = 1000.0 / np.array(temps) # 1000/K
           ln_D = np.log(np.array(diff_coeffs))
           
           slope, intercept, r_val, _, _ = linregress(inv_T, ln_D)
           
           # ln(D) = ln(D0) - (Ea / kB) * (1/T)
           # slope = -Ea / kB * 1000 (since we used 1000/T)
           # Ea = -slope * kB / 1000
           kB = 8.617e-5 # eV/K
           Ea = -slope * kB * 1000.0
           
           plt.figure()
           plt.plot(inv_T, ln_D, 'o')
           plt.plot(inv_T, slope * inv_T + intercept, '-')
           plt.xlabel("1000/T (K⁻¹)")
           plt.ylabel("ln(D [cm²/s])")
           plt.title(f"Arrhenius Plot\nActivation Energy Ea = {Ea:.3f} eV")
           plt.savefig("arrhenius_plot.png")
           print(f"Saved arrhenius_plot.png. Calculated Ea: {Ea:.3f} eV")
           
           return Ea
       else:
           print("Not enough points for Arrhenius plot.")
           return None

   if __name__ == "__main__":
       # Usage: python analyze_diffusion.py
       analyze_diffusion("results", species_element="Li")
   ```

6. **Verify and Refine**:
   - Check the `msd_plot.png` to ensure MSD is linear (diffusive regime). If flattening, simulation time might be too short or material is trapped.
   - Check `arrhenius_plot.png` for linearity.
