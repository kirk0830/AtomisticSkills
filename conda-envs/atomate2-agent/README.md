# Atomate2 Agent Environment

This environment supports Atomate2 workflows.

## Quick Installation
Most users should use the simplified installation script, which installs only the necessary core packages:

```bash
bash install.sh
```

This installs:
- python 3.10
- atomate2
- pymatgen
