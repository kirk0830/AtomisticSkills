# Smol Agent Environment

This environment supports the Smol cluster expansion package.

## Quick Installation
Most users should use the simplified installation script, which installs only the necessary core packages:

```bash
bash install.sh
```

This installs:
- python 3.10
- smol
- pymatgen
- atomate2
