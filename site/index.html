Three-Level Research Hierarchy

A composable framework for AI-driven atomistic research. Chain modular skills across MLIPs, DFT, drug discovery, and generative AI — in any coding copilot.

Quick Start

Get Started in Minutes

Install only the environments you need for your research tasks.

bash 
# Clone the repository
git clone git@github.com:bowen-bd/AtomisticSkills.git && cd AtomisticSkills

# Auto-patch paths for your machine
python configure_mcp.py

# Install only the envs you need
bash conda-envs/base-agent/install.sh
bash conda-envs/mace-agent/install.sh    # MACE models
bash conda-envs/matgl-agent/install.sh   # MatGL / CHGNet

# Add credentials to ~/.atomistic_skills.yaml
echo 'MP_API_KEY: your_key_here' >> ~/.atomistic_skills.yaml
